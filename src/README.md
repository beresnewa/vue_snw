Соцсеть для котиков :3

Команда запуска приложения: npm run serve

Для теста приложения лучше авторизовываться под этим аккаунтом: login: ann, password: 123, т.к. он не пустой 

Регистрация / Авторизация
Страница профиля "Profile" отображает профиль авторизованного пользователя:
    кнопка "Edit profile": открывает страницу для загрузки файлов: аватарки и фотографий для профиля;
    кнопка "Followers" покажет список подписчиков данного пользователя;
    кнопка "Subscriptions" покажет список подписок данного пользователя, кнопка "отписаться" удалит подписку;
    загруженные публикации можно удалить нажав на крестик в углу фото;
Страница пользователей "Users" отбражает список всех зарегистрированных пользователей:
    чекбокс "Sort by age" отфильтрует всех пользователей по возрасту по возрастанию;
    "поиск" отфильтрует пользователей по введенному имени;
    кнопка "Subscribe" добавит пользователя в подписки, а также авторизованный пользователь будет добавлен в подписчики к этому пользователю;
    кнопка "Show more" будет догружать по 3 пользователя;
Кнопка "Logout" выйдет из аккаунта;
